# Platform Filters For Cosmetic QOL

Filters and changes platform served cosmetics for uBlockOrigin


## How to use
1) From ```uBlockOrigin``` settings go to ```Filter lists```  
2)  Scroll down to ```Import``` and copy the filter Raw URL
